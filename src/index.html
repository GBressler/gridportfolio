Fireweed Photography
Repo Demo

Skills & Tools Used

Some of the skills and tools that I used in making this site were: WordPress 5.2, cPanel and MySQL.

Objective

For this site, my goal was to create a site that would cover all the bases for a photography business, while at the same time looking picture-perfect.

Results

In the end, the Fireweed Photography site is a site that would make any owner proud. The site includes a nicely organized group of posts and pages that are searchable and can be quickly filtered by Topics or Categories. Users can comment in the comment section. Each page has a footer integrated with Google Maps to help clients find the store. The site is integrated with Google Analytics so the owners can learn more about the visitors to better market their business. There is a portfolio section for the business to show off a few of their excellent pieces. Finally, there is a working and contact form and a larger map on the Contact Page.

Catch of the Day
Repo Demo

Skills & Tools Used

For this project I used quite a lot...including: HTML, CSS, Javascript, React & React Router, Local Storage, Facebook API Authentication and Firebase.

Objective

The idea for this app is to allow a store or restaraunt owner to have more control of their inventory and make adjustments in real time.

Results

This final product is a web app that lets clients place orders and the app then makes a detailed invoice which updates live as orders are placed. The owner can change prices to meet demand, add & update inventory and finally mark an item as "Sold Out" when the last of a specified item is sold. The app uses a combination of local browser storage and Google's online database Firebase...and React...because it's so hot right now.
Robofriends
Repo Demo

Skills & Tools Used

With this project I used Tachyons 4.11, React 16.8 and the Robohash API.

Objective

The idea of this project is to be able to search through the vast sea of your Robofriends.

Results

The final site took a file full of information about our Robofriends, read it and produced an image for each Robofriend in the file. Each Robofriend has a nice animation and we have a search bar for a quick search and filtering of our robots.
Laptop Store
Repo Demo

Skills & Tools Used

With this site I used HTML, Javascript and Node.

Objective

This was largely an experiment to get a small site up and running use only Node and the basics.

Results

The end result of this project is a site that reads in inventory and product descriptions from a file and inserts the information into card templates. The templates are displayed neatly with a brief description of each laptop and a link to a page featuring the laptop. All routing is done completely through Node.
